# 📐 Направляющие для Chrome DevTools

Перетаскиваемые направляющие линии на любой странице.

Попробовать без установки можно на [демонстрационной странице](https://kulserhub.github.io/Chrome-DevTools-Guides/) — там скрипт уже запущен.

## Установка

1. Открыть **DevTools** (F12) → вкладка **Sources** (Источники) → панель **Snippets** (Фрагменты)
   (если панели не видно — стрелочка `»` слева)
2. "**+ New snippet**" (+ Новый фрагмент), задать название, вставить код из `snippet.js`
3. `Ctrl + S` — сохранить

## Запуск

- Открыть **DevTools** (F12) → вкладка **Sources** (Источники) → панель **Snippets** (Фрагменты)
- Правый клик по фрагменту → **Run** (Запустить), либо `Ctrl + Enter` когда он открыт
- Или нажать: `Ctrl + P`, ввести символ `!` и имя фрагмента, `Enter`

## Управление

| Клавиша | Действие |
|---|---|
| **V** | Вертикальная направляющая под курсором |
| **H** | Горизонтальная направляющая под курсором |
| Перетащить | Двигать направляющую |
| Двойной клик по направляющей | Убрать одну направляющую |
| **Esc** (два раза подряд) | Убрать все направляющие |

В полях ввода клавиши V и H не срабатывают, чтобы не мешать печатать.

После запуска фрагмента можно закрыть DevTools клавишей `F12` — направляющие останутся на странице до её перезагрузки.

Направляющие сохраняются отдельно для каждого сайта и возвращаются при следующем запуске фрагмента.

## Настройка

В начале кода:

```js
const GRAB = 10;       // сколько пикселей захватить с каждой стороны направляющей
const COLOR = "#0ff";  // цвет направляющей
```

Направляющие всегда ровно 1px, но хватать их можно в радиусе `GRAB` — так проще попасть мышкой.
