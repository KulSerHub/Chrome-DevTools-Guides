📐 Направляющие для Chrome DevTools

Скрипт уже запущен на этой странице — ставьте направляющие прямо сейчас, ничего устанавливать не нужно.

  • Клавиши V и H добавляют вертикальные и горизонтальные направляющие
  • Направляющие можно передвигать мышью
  • Двойной клик по направляющей — убирает её
  • Двойное нажатие клавиши Esc — убирает все направляющие
  • Направляющие сохраняются для этого сайта и возвращаются при перезапуске
Это демонстрационная страница. Здесь скрипт подключён заранее, чтобы вы могли попробовать направляющие без использования фрагментов Chrome DevTools. Чтобы они работали на любом сайте, установите фрагмент по инструкции:
https://github.com/KulSerHub/Chrome-DevTools-Guides

Найдите неровности

Большинство карточек выровнено по сетке, но несколько выбиваются. Поставьте направляющие вдоль края блока — и расхождение станет видно.
